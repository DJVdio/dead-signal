# Dead Signal 正式音频库

这里的音频是项目自有、确定性离线渲染的正式资源，不依赖网络服务或第三方采样包：

- `music/`：7 首 24 秒自适应配乐，分别服务菜单、营地昼夜、探索、交战、尸潮和结局。
- `ambience/`：5 条 16 秒环境循环，覆盖营地昼夜、户外、室内和下水道。
- `sfx/`：22 类一次性音效，每类 3 个独立变体；运行时轮换变体并施加极轻微音高变化。

全部 master 为 44.1 kHz、16-bit、双声道 WAV。音乐与环境声由运行时设置完整文件循环；音效不循环。

可复现源为 `tools/generate-audio-assets.py`。该脚本需要 Python 3 和 NumPy，重新执行会确定性覆盖全部 78 个 master。运行时优先加载这里的文件，只有资源缺失或导入失败时才退回旧的程序合成器，避免静默失声。

音频只承担表现，不改变噪音传播半径、武器结算、丧尸感知或其他玩法规则。
