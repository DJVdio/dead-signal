# 结局 CG 资源说明

本目录保存结局面板使用的 16:9 faux-isometric 像素风背景。图片由内置图像生成模型生成，叙事条件仍以设计文档和运行时 `StoryFlags` 为准；CG 只表现既有结局，不新增剧情事实。

| 文件 | 使用场景 | 生成提示摘要 |
| --- | --- | --- |
| `military-escape.png` | 军方撤离结局 | 孤身幸存者离开损毁营地，远处为匿名军人和撤离载具；不表现军方动机或具名新角色。 |
| `horde-escape.png` | 尸潮逃生结局 | 孤身幸存者从庞大的普通丧尸群前逃离；不加入精英丧尸或新剧情线索。 |
| `family-win.png` | 全员团聚结局 | 现有队伍在黎明时抵达放下的桥并被接应；不新增角色关系或对白。 |

共同风格约束：低饱和、阴郁末世、有限色盘、硬边像素块、无文字、无徽标、无 UI；主要人物和叙事焦点避开画面中央的结局文字安全区。

实机验收时检查不同窗口比例下的裁切、文字对比度和角色辨识度。
