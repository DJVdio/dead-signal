# 需求 - 1 - 隐匿岗哨对抗
> 状态：待开发
> 优先级：P1
> 工作量：大
> 前置依赖：无

## 背景

夜防已经有“守卫警觉 vs 袭击者隐匿”的纯逻辑和运行时流程，敌营也已经有岗哨的视野、光照、距离和遮挡基础判定。但夜防运行时把袭击者服饰隐匿固定为 0，敌营岗哨没有消费玩家实际服饰隐匿，也没有接通 Rat 的三级黑暗隐匿与破隐先手效果，因此 Wiki/配置中的能力没有完整落地。

## 目标

在不新增地图几何、不改变既有默认路径的前提下，完成夜防服饰隐匿接线、敌营岗哨的玩家隐匿对抗，以及 Rat L3 的黑暗隐匿和一次性破隐先手；以 TDD、运行时接线护栏和全量构建证明系统已落地。

## 功能点

### FP-1 夜防读取实际服饰隐匿

- 行为描述：夜防袭击者的隐匿计算使用其实际装备服饰的隐匿值汇总，不再传固定 0。
- 输入：袭击者实际 `DefenderArmor`/服饰名称、当前光照、最近守卫距离、掩体权重。
- 输出：传入 `NightWatchContest.ComputeStealth` 的实际服饰隐匿值和对应对抗结果。
- 异常：服饰为空或未登记时隐匿贡献为 0；不得因缺少登记导致运行时异常。

### FP-2 敌营岗哨接入玩家隐匿对抗

- 行为描述：在现有岗哨视野锥、范围、局部光照、携带光源和遮挡通过后，使用玩家探索隐匿与岗哨警觉进行发现判定。
- 输入：玩家位置和光照、真实穿戴服饰、距离、遮挡/掩体、岗哨扫描参数。
- 输出：未通过对抗时保持未发现；通过对抗时进入现有敌对响应/战斗流程。
- 异常：视野基础条件不满足时不得凭空发现；未启用该对抗的既有敌人保持原行为。

### FP-3 Rat L3 黑暗隐匿

- 行为描述：Rat 的三级黑暗隐匿效果作用于敌营探索隐匿计算，沿用现有 +50% 配置，以乘算方式应用。
- 输入：Rat 身份/专属效果、敌营探索中的基础隐匿值。
- 输出：拥有 Rat L3 时黑暗隐匿相关项按 1.50 倍计算；其他角色和其他场景不变。
- 异常：没有 Rat L3 或不在敌营探索隐匿判定中时不得应用加成。

### FP-4 Rat L3 一次性破隐先手

- 行为描述：Rat 在尚未被发现时对敌对目标发起第一次攻击，获得一次 +35% 伤害；攻击后或被发现后结束隐匿先手状态。
- 输入：攻击者身份、是否仍未被发现、是否为第一次敌对攻击、原始伤害。
- 输出：满足条件时伤害乘以 1.35 并标记已暴露/已使用；其他攻击返回原伤害。
- 异常：已暴露、已使用、非 Rat、非敌对攻击或没有该专属效果时不得加成；不得重复触发。

### FP-5 测试、接线与回归

- 行为描述：为纯逻辑公式、实际服饰读取、岗哨运行时接线和 Rat 一次性状态补齐测试，并运行项目门禁。
- 输入：边界数据、真实角色/装备、既有夜防/探索场景。
- 输出：测试能证明需求链路已接通，完整 .NET 测试与 Godot C# Release 构建通过。
- 异常：若接线退回固定常数、Rat 加成跨场景泄漏或既有行为漂移，测试必须失败。

## 验收标准

- [ ] AC-1 夜防运行时传入袭击者真实服饰隐匿值，换装差异可被测试观察到。
- [ ] AC-2 未登记服饰的隐匿贡献为 0，且不会抛出异常。
- [ ] AC-3 敌营岗哨保留原有视野锥、范围、光照、携带光源和遮挡条件，并在其后接入玩家隐匿对抗。
- [ ] AC-4 隐匿目标可在对抗失败时保持未发现，暴露目标可在同条件下被发现并进入现有响应流程。
- [ ] AC-5 Rat L3 黑暗隐匿在敌营探索中乘算为 1.50 倍，夜防和战斗引擎不受影响。
- [ ] AC-6 Rat L3 破隐先手只在未发现状态下第一次敌对攻击乘算为 1.35 倍。
- [ ] AC-7 破隐先手触发后、Rat 被发现后、非 Rat、非敌对攻击和无该效果场景均不重复加成。
- [ ] AC-8 普通丧尸、未启用新对抗的敌人、既有 `NightWatchContest`/`SentryLogic` 行为和 Sim 基线无回归。
- [ ] AC-9 纯逻辑测试、完整 `~/.dotnet/dotnet test` 和 Godot C# Release 构建通过。

## 非功能需求

- 规则逻辑保持纯 C#、可单元测试，不把 Godot 空间对象引入战斗引擎。
- 百分比加成使用乘算，随机判定使用可注入随机源并保持可复现。
- 数据只从真实装备/既有配置读取，禁止为接线复制一份手写数值。
- 新字段默认值必须保持既有调用方行为，普通丧尸和未启用探索隐匿对抗的敌人零回归。
- 不新增地图几何、巡逻换班、警戒等级、探索控制或剧情内容。

## 影响面提示（产品视角）

- 夜防玩家会首次感知到袭击者服饰对隐匿的实际影响。
- 敌营探索的潜行结果会受角色服饰、光照、距离和遮挡共同影响，岗哨不再只按基础视野直接发现。
- Rat 的三级能力会形成明确的“更容易接近 + 第一次攻击更痛”的专属潜行身份，但不能连续攻击重复享受。
- 本需求不改变 Goldfinger 地图布局；若后续需要真正的潜行路线、门后区域或巡逻网络，应另立需求。
