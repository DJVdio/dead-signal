# 隐匿岗哨对抗

## 1. 文档信息

- 需求编号：1
- 创建日期：2026-07-18
- 状态：待开发
- 优先级：P1
- 目标版本：当前开发版本

## 2. 背景与问题

游戏已有两类相关的对抗基础：夜间营地防守中的“守卫警觉 vs 袭击者隐匿”，以及敌营中的岗哨视野、光照和遮挡判定。但目前两条链路都没有完整消费实际装备和角色效果：夜防袭击者的服饰隐匿值在运行时被固定为 0，敌营探索也没有把玩家的服饰隐匿和 Rat 的三级效果接入岗哨发现流程。

结果是玩家在数据层配置的服饰差异、黑暗环境和 Rat 的专属效果，不能稳定地影响“是否被岗哨发现”或“是否能打出破隐先手”。

## 3. 目标与非目标

### 3.1 目标

1. 夜防对抗读取袭击者实际穿着的服饰隐匿值。
2. 敌营岗哨在现有视野、光照、距离和遮挡基础上，增加玩家探索隐匿与岗哨警觉的对抗。
3. 接通 Rat 的三级黑暗隐匿和破隐先手效果，并保证效果只在规定场景和次数内生效。
4. 用纯逻辑测试和运行时接线测试证明上述链路已落地，且不改变未启用新能力的既有行为。

### 3.2 非目标

- 不新增 Goldfinger 地图的门、围栏、潜行路线或其他关卡几何。
- 不实现岗哨巡逻、换班、全局警戒等级或新的敌营编队。
- 不补做探索队伍的点击移动、搜刮交互等其他探索控制需求。
- 不擅自调整 Wiki 中已有服饰、岗哨、Rat 或战斗数值。
- 不新增剧情文本、角色关系或 authored 内容。

## 4. 用户场景

### 场景 A：夜间袭击者穿着不同服饰

玩家在夜间营地防守时面对袭击者。相同光照、距离和掩体条件下，袭击者穿着不同服饰时，隐匿值应按实际服饰变化；未登记隐匿值的服饰不产生额外加成。

### 场景 B：玩家接近敌营岗哨

玩家控制探索队接近敌营岗哨。岗哨仍先受原有视野锥、视野范围、局部光照和遮挡影响；在具备可见条件后，再由玩家隐匿与岗哨警觉决定是否被发现。玩家处于黑暗、距离更远或有遮挡时更难被发现。

### 场景 C：Rat 的专属潜行与先手

Rat 在敌营探索中享受三级黑暗隐匿效果。Rat 在尚未被发现时对敌对目标发起第一次攻击，可获得一次破隐先手伤害加成；攻击后隐匿状态结束，后续攻击不重复获得该加成。

## 5. 功能需求

### 5.1 夜防服饰隐匿接线

- 夜防袭击者的隐匿计算必须读取该袭击者实际装备的护甲/服饰名称。
- 服饰隐匿值沿用现有数据登记；只读取实际穿戴内容，不复制或手写另一份数值。
- 未登记服饰的隐匿贡献为 0。
- 守卫警觉公式、夜防距离/光照/掩体规则和随机顺序不因本需求改变。

### 5.2 敌营岗哨隐匿对抗

- 岗哨继续使用现有视野锥、视野范围、光照、携带光源和遮挡作为基础可见性条件。
- 目标满足基础可见性条件后，使用统一的隐匿对抗逻辑计算玩家探索隐匿与岗哨警觉的结果。
- 玩家隐匿读取真实穿戴服饰；不同服饰应能形成可测试的结果差异。
- 岗哨保持岗哨类型和现有警戒/扫描参数，不引入新的地图几何或未授权的自动巡逻规则。
- 未启用探索隐匿对抗的敌人、普通丧尸和现有默认调用路径保持原行为。

### 5.3 Rat 三级效果

- Rat 的“黑暗隐匿 +50%”只作用于敌营探索隐匿计算，不作用于夜防袭击者、战斗引擎或其他阅读/制作效果。
- Rat 的“破隐先手额外 +35% 伤害”只在 Rat 尚未被发现时发起的第一次敌对攻击生效一次。
- 破隐先手触发后，Rat 视为已暴露；同一轮/同一潜入状态下的后续攻击不得再次获得该加成。
- 没有 Rat 三级效果、已经暴露或非敌对攻击不获得该加成。

### 5.4 可验证性与回归

- 纯逻辑层覆盖隐匿计算的黑暗、服饰、距离、遮挡和 Rat 加成边界。
- 运行时接线测试确认夜防和敌营岗哨都传入真实装备/角色效果，而不是固定常数。
- 保留现有 `NightWatchContest`、`SentryLogic`、战斗和探索相关测试，并补充必要的回归护栏。

## 6. 交互与状态

1. 敌营岗哨发现玩家后，沿用现有敌对响应和战斗流程；本需求不新增 UI。
2. 玩家未被发现时，不弹出发现提示、不触发敌对响应；沿用现有探索画面状态。
3. Rat 的破隐先手状态应是一次性状态，且在触发攻击或被发现时结束，避免跨场景或跨战斗残留。

## 7. 数据与配置

- 服饰隐匿值以现有 `Apparel`/夜防配置和 Wiki 数据为准。
- Rat 三级效果沿用现有配置中的 1.50 倍黑暗隐匿与 1.35 倍破隐伤害，不在本需求中改数值。
- 新逻辑使用可注入随机源（如需要随机判定），不得引入不可复现的全局随机。

## 8. 验收标准

- [ ] 敌营岗哨在现有视野/光照/距离/遮挡基础上接入玩家隐匿对抗。
- [ ] Rat L3 黑暗隐匿在敌营探索中按 1.50 倍生效，其他场景不受影响。
- [ ] Rat L3 破隐先手在未发现状态下第一次敌对攻击按 1.35 倍生效一次，之后不重复。
