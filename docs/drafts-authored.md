## §1. 军方屠杀动机 —— 3 个候选设定方案（用户选一个作正史）

> 正史约束：军方在**第 3 天白天**来袭、屠杀营地留守者（§3 结局②）；动机是"背叛"主题的支点，此前一直是 authored 留白。以下三案各给「世界观阐述 / 对玩家的信息揭示方式 / 叙事代价」，**由用户选一个定为正史**，其余弃用。三案都不需要角色开口解释——都靠残片、痕迹、反讽让玩家自己拼出来。

### 方案 A：净化协议（清剿感染风险）

- **世界观**：军方早已不再区分"感染者"与"健在幸存者"。收容区容量在灾变初期就崩了，高层签发了"净化协议"——凡是无法接收的区域，一律视为潜在感染源就地清除。那段循环救援广播不是救援，是**筛选诱饵**：回复即暴露坐标，坐标即被登记为"待清除的潜在感染点"。来的人只是照单执行的士兵，命令从更高的地方来。
- **信息揭示**：军人尸体/遗物上的一页作战简令残片（"区域标记：待净化"）；无线电呼号里反复的"净化(purge/sterilize)"字样；营地收音机里那段"重建收容区、等待接应"的广播——玩家事后重听，字字都是反讽。可选：一份军方备忘同型的 lore 物品（与日记 A/B 同机制，可拾取带回读）。
- **叙事代价**：把军方降为**系统性、非人格的恶**——"背叛"更像一台冷机器而非某个人的选择，悔恨感最强（是玩家亲手按下求救键招来灭顶），但军方失去人性面孔、较扁平，玩家的恨无处落点。

### 方案 B：救援是幌子·资源掠夺

- **世界观**：所谓"收容区"其实早已沦陷、自身难保。还在播的救援广播，是打着军方旗号的残部/溃兵设下的**诱饵**——回复坐标暴露的，是一个还有物资、还有活人的营地。他们来的目的从不是救人，是抢光补给、清掉见证者，然后转去下一个信号点。
- **信息揭示**：来袭者不是齐整的正规编制（番号混杂、装具东拼西凑）；现场"抢完就走"的痕迹——仓库被搬空、值钱的没了、人却没被带走；没有任何救援器械（没有担架、没有运输、没有医疗），只有枪。
- **叙事代价**：军方"救援"本身是骗局，主题从"制度性冷酷"滑向"末世无政府、谁都靠不住"；削弱了"官方救援"作为希望锚点的分量（玩家会更早对广播免疫），但最贴近掠夺现实，也让"呼叫南方营地"作为唯一生路更成立（对照：南方也未必可信，但至少没骗你来杀）。

### 方案 C：命令与执行者的失控（误判/私通）

- **世界观**：军方本欲救援。但"回复坐标"在崩坏的指挥链里被误读——或被一个失控的指挥官判为"敌对武装据点",或坐标恰好落进了私通劫掠者、拿平民换补给的军官手里。来的人带着**错误的交战命令**，等他们踏进营地、意识到这里只是几个平民和一锅凉掉的饭时，已经太迟。
- **信息揭示**：军人临死前的一句迟疑残话（"……这里不是……我们收到的不是这个……"）；作战命令残片与眼前现实的落差；一枚军牌 + 一封没来得及寄出的家书——加害者也是别人的儿子/父亲。
- **叙事代价**：给"背叛"最多的道德灰度（加害者也是齿轮、也会后悔、也是受害者），最不黑白、最耐咀嚼；但也**削平了"背叛"的锋利**——玩家很难把恨集中到一个对象上，悲剧更偏"误会与失控"而非"出卖"，与"你亲手招来了这一切"的自责主题略有拉扯。

> 备注：三案不互斥于机制层（军袭事件本体、时序、留守/外出判定都一样），只影响揭示文本与 lore 物品内容。选定后，§2 军袭分镜的"署名"拍（拍 4）与 §4 结局② CG 的旁白口径随之定稿。

## §2. 军袭事件分镜草案（第 3 天白天·探险队视角归来所见）

> 口径（§3 结局②）：军袭在白天发生、屠杀留守者；**外出探险队幸存归来**，见营地覆灭只剩自己几人；游戏不强制结束。**无幸存者口述**——全靠尸体与痕迹讲故事。台词极少，能省则省。以下 7 拍（含 1 可选），顺序可调。

- **拍 1｜远景·不对劲**：世界地图归营的过场落地，镜头先给营门——本该有守卫的岗哨空着，**营门从外面被拉开着**。风。没有一个人影。
- **拍 2｜脚下·制式弹壳**：进营，镜头压低。地上一排排弹壳，不是营地那几把杂牌枪的口径——是**制式步枪**的，排布出训练有素的射界。有人站成一线开过火。
- **拍 3｜长桌·排着打**：聚餐的长桌边，留守者伏在各自的位子上。没有四散奔逃的散乱——是被**排着**打的。锅里的饭还在，早就凉透了。（此拍最狠，克制到底，不给特写血腥，只给"各自的位子"。）
- **拍 4｜唯一的署名·却什么都不解释**：墙上/门上留着唯一一处"署名"——一枚军方标识、一张制式通告残片、或一行喷漆记号（**内容随 §1 选定动机定稿**）。它证明是谁干的，却对"为什么"只字未提。（这一拍承载"动机留白"的主题重量。）
- **拍 5｜收音机·最刺的反讽**：营地那台只能收的收音机还开着，军方循环救援广播仍在一遍遍重复——"……请保持冷静，就地固守……等待接应……"。声音盖过全场的死寂。
- **拍 6｜布鲁斯/铁盒·非人视角的哀悼**（择一，视存档状态）：
  - 若布鲁斯在营且活着：它从瓦砾后一瘸一拐钻出来，绕着道格常坐的空位子低鸣，不肯离开（接 `meal_bubbles.json` 道格死亡后旁观组的同一意象）。
  - 或：中庭那只被翻出又扣回原处的铁盒旁（接 `egg_courtyard_family` 彩蛋），压着新的、还没干的血。
- **拍 7｜院中·沉默与俯瞰**：幸存的探险队几人站在院子正中，没有台词——或只给一句克制到底的："……门是从外面开的。" 镜头缓缓拉高，俯瞰整片营地：就是开局玩家熟悉的那块地，如今静止。
- **拍 8｜可选·游戏继续**：HUD 角落，尸潮倒计时悄然回到画面——还在走。淡出。玩家带着剩下的人，接着往下活。

> 关键台词/环境叙事清单（供裁剪）：①探险队沉默为主，全片至多 1 句（"门是从外面开的"／或完全无台词）；②唯一的"声音"是收音机循环广播的反讽；③一切因果交给弹壳排布、排坐的尸位、唯一的署名、凉掉的饭——**让玩家自己读**。

## §3. 南方营地 / 考验 / 峡谷南逃流程草案（结局③·唯一生路）

> 口径（§3 结局③）：取得电台发出设备 → 呼叫南方营地求救 → 请求他们**临时开放"前往峡谷的路"** → 完成**考验** → 带**一小部分物资**向南逃离（二号大地图伏笔）。**南逃不冻结时限，须抢在第 40 天尸潮前完成**（主 agent 默认，用户可否决）。以下为接触对话 + 考验候选形态 + 开路 + 启程文本。

### 3.1 接触对话（呼叫南方后·电台那头，draft·待用户细化）

> 基调：南方警惕、不信任、要价；不温情。往返约 8 句。

- 南方（杂音里）："……收到。你们是哪个点的？说清楚，别浪费我的电。"
- 我方："城北的一处营地。北面尸潮在南下，我们撑不了多久，请求接应。"
- 南方："接应？我们自己都嫌人多。北面的事我们知道——那也是我们不想开门的理由。"
- 我方："我们不白来。只求你们开一次'前往峡谷的路'，我们自己走。"
- 南方（沉默几秒）："……路可以谈。但不是白开。这边的规矩：想过去，先证明你们值这个险。"
- 我方："什么条件？"
- 南方："等着。有活儿给你们。办成了，路给你们留三天。办不成——就当没通过话。"
- 南方："还有，能带走的，只有背得动的那些。别指望把家搬过来。……就这样。滋——"

### 3.2 考验 —— 2~3 个候选形态（用户挑一个/可组合）

- **形态 A｜护送 / 接应**：南方有一小队人（或一名信使/一批药）困在半路，要玩家派队去接到某点，或清掉堵在峡谷口的一处威胁（丧尸巢 / 劫掠者哨卡）。
  - *代价*：一场高难度探索关，可能折人；与"夜里守备名册变薄"的昼夜班权衡叠加。
- **形态 B｜献粮 / 投名状**：南方物资也紧，要玩家先交出一定量物资 / 药品 / 弹药，作"过路费"兼诚意，证明不是来分食的累赘。
  - *代价*：经营层硬性抽血，与 40 天时限抢时间双重挤压——你越拖越交不起。
- **形态 C｜武力证明**：南方只认强者，设一场"证明"——守住一波袭击 / 派人单独闯过一段路，活着回来才开门。
  - *代价*：得把队里最能打的押上去，输了赔命，赢了当晚夜防更空。
- **组合建议（可选）**：先 B 献粮换取接触资格，再 A 护送 / C 证明换开路——两段式，把时限压力拉满。

### 3.3 开放峡谷之路

- 考验通过 → 南方"临时开放前往峡谷的路"：世界地图新增一个**临时"峡谷通道"点**（限时，如启程窗口三天，拟定待调）。
- 携带上限：只能带**一小部分物资**（背得动的量，具体上限拟定待调）——逼玩家取舍，为二号地图的"轻装重开"定调。

### 3.4 南逃启程文本（draft·待用户细化）

> 启程前一段旁白 + 一句二号大地图伏笔。克制、不圆满。

> 你们把能带的都捆上了——只有背得动的那些。剩下的，留给这座城，和它身后追来的东西。
>
> 峡谷的另一头是什么，南边的人不肯说，只说"比这里强"。你不知道该不该信。可留下来，只有一种结局。
>
> 队伍走出营门时，没人回头。身后，倒计时还在走。

- **二号大地图伏笔句（南方营地口吻，择一）**："峡谷南面不是终点，是另一个开始。" / "过了峡谷，就没人认得你们是从哪儿逃出来的了——那才是活路。"

## §4. 三结局 CG 文案草案

> 三条正史结局的 CG 正文属用户手写剧情。以下为初稿，基调各异但都克制。§3 结局矩阵已定"全员战死进 CG、以逐个死去与活着的名字收束全篇"。

### 4.1 结局①：第 40 天尸潮全灭（悲壮终局·呼应瞭望台所见）

> 呼应 `LookoutSighting`《正北方，天际线尽头》——望远镜里那片"阴云"终于漫到脚下。无限围攻，逐个死去。

> 望远镜里的那片阴云，终于漫到了脚下。
>
> 它不是云。你们早就知道。
>
> 一波，又一波。你们把门顶了一次又一次，把还能开火的人一个个补到缺口上。没有援军。从来没有。
>
> （名单逐个划去——活着的名字一个接一个熄灭）
>
> 你守住了这里，直到最后一个人。
>
> 这样做，是对的吗？

### 4.2 结局②变体：军袭后全员在营 = 全灭（背叛·全灭变体）

> 与①同为全灭 CG，但成因不同——不是尸潮，是**你自己按下的求救键**招来的人（仅当军袭时全员恰在营、无人外出）。旁白更冷、更反讽；背景是军方循环广播。

> 你按响了求救的信号。
>
> 他们真的来了。
>
> （收音机还在循环："……请保持冷静，就地固守……等待接应……"）
>
> 没有人在外面。这一次，没有人回得来。
>
> （名单逐个划去）
>
> 你等来的，是你呼叫的那一方。

### 4.3 结局③：南逃成功（唯一生路·留悬念不圆满）

> 不是胜利，是"还没死"。留悬念、开放，一丝微光；呼应 dead signal 主题，为二号地图伏笔。

> 你们逃出来了。
>
> 活下来的没剩几个。带走的物资，撑不了多久。
>
> 峡谷的风，比城里干净一些。前面是什么，没人知道。
>
> 至少，信号还在——
>
> 只是这一次，是你们自己，往前走。

## §5. 道格入队剧情草案 —— 2 个候选时机 / 情境（供用户挑）

> 口径：道格 = 普通可入队幸存者（持手枪、打不太准）；布鲁斯 = 其犬类伙伴（高闪避 / 高移速 / 低伤害缠斗、能站岗、能随队出探索）。人狗相依为命本身即情感锚点。入队时机 / 性格 / 台词属用户手写，以下为初稿供挑。**首次相遇对话 6~10 句 + 布鲁斯的表现**。

### 候选甲：探索途中相遇（住宅区 / 河边小屋等目的地）

- **情境**：探险队在某目的地一处角落，撞见道格和布鲁斯被丧尸逼到死角——道格护着受伤的狗（或狗护着弹尽的人），已经打光子弹。玩家解围后，道格请求带上他和狗。
- **布鲁斯的表现**：先是低吼、拦在道格身前戒备；道格一个手势 / 一声"布鲁斯"，狗立刻噤声退到他腿边——人狗默契的第一印象，一个动作胜过十句台词。
- **首次相遇对话（draft·6~10 句）**：
  - 道格："别开枪——枪里也没子弹了。我就是想让你们看清楚，我不是那东西。"
  - 道格（拍拍狗）："这狗叫布鲁斯。它比我有用。"
  - （布鲁斯低吼，道格搭手，狗安静下来）
  - 道格："它认人。刚才没扑你们，就是它信得过。"
  - 道格："我叫道格。会点粗活，枪也能打——就是打不太准。"
  - 道格："你们要是嫌多一张嘴喂狗……那我也不去了。它走哪，我走哪。"
  - 玩家（收留）：……
  - 道格："……行。那从今天起，我们俩这条命，搭你们这儿了。"
  - （布鲁斯蹭了蹭道格的手背）

### 候选乙：营门来投（清晨 / 黄昏，营门外一人一狗）

- **情境**：某个清晨或黄昏，营门外来了一人一狗。道格敲门求收留，坦白自己没什么本事，但布鲁斯能顶个岗、能闻出夜里靠近的东西（呼应站岗 75% / 夜间嗅觉设定）。玩家选收留 / 拒。
- **布鲁斯的表现**：隔着门缝先探出头，凑近闻玩家的手——接纳与否的信号交给狗的鼻子，而不是台词。
- **首次相遇对话（draft·6~10 句）**：
  - 道格（隔门）："别紧张。就我跟我的狗，没别人。"
  - 道格："我知道这年头开门是拿命赌。所以我先把话说清楚——我不白吃饭。"
  - 道格："狗叫布鲁斯。它能站岗，能替你们守夜。夜里有东西摸近，它比谁都先知道。"
  - （门缝里，布鲁斯探头，凑近闻了闻玩家的手，没有龇牙）
  - 道格："它闻过你了。它要是信不过，早叫了。"
  - 道格："我呢，枪法一般，力气还有。留不留，你们说了算。"
  - 玩家（收留）：……
  - 道格："谢了。……布鲁斯，进去吧。以后这儿就是家了。"

> 备注：两候选的入队时机（第几天 / 触发条件）与道格性格细节留用户拍板。聚餐里的人狗日常 / 死亡气泡已落 `meal_bubbles.json`（走 `bruce_present`/`bruce_dead`/`doug_dead` 门控），可与本节入队剧情前后呼应。
