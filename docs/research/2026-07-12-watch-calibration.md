# 夜防发现率矩阵解析校准（night-response 修复后复扫）

模型：逼近带 {150,90} 各掷一次·跨带累积；疲劳视锥×(0.85,0.9) + 听力项×0.4（双路径，night-response 修复后）。
权重：视力 1/听力 0.6(半径 220)　潜行 暗1/距0.6(参考 300)。

## 全营发现率（南向逼近，跨带累积，带 {150,90}）
| 场景 | 发现率 | 目标 | 判定 |
|---|---:|---|---|
| 裸营(1守卫·无火·无疲劳) | 48 % | 35-55% | ✓ |
| 满配·**分散布防**(3守卫散在3岗+火堆) | 65 % | 80-90% | 偏低（岗位分散只1守卫在南轴感知内） |
| 满配·**覆盖逼近轴**(3守卫盖南轴+火堆) | 85 % | 80-90% | ✓ |
| 覆盖+疲劳(视锥×0.85/0.90+听力×0.4) | 67 % | 15~25pp | Δ=18pp ✓ |

## 结论
- **裸营 48% ✓**（带 {150,90} 定值，落 35-55%）。
- **满配 80-90% 取决于守卫覆盖**：分散布防 65%（3 岗位散在东/西/南，任一逼近轴只 1 守卫在感知内）；覆盖逼近轴 85% ✓。→ 需 night-response 布防站位（守卫朝威胁方向集中/岗位重叠盖大门），非纯数值可调。[DECISION]
- **疲劳劣化达标（Δ=18pp，落 15~25pp）**：视锥 ×0.85/0.90 单路径饱和(~4pp) → night-response 修复补一道听力项 ×0.4（NightRaidLogic.FatigueAdjustedAlertness，运行时与本 Sim 同调单一真源）。听力是满配检测大头，削之即打破地板饱和。[DECISION-RESOLVED]
