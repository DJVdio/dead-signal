## ②b 群战：1 人 vs N 只丧尸（霰弹枪的定位主场）

Arena 是**无空间**模型：8 颗弹丸全落在同一只丧尸身上。真实游戏里锥形散射会让一枪同时扫到多只（Godot 侧 8 枚独立弹丸各自碰撞）——故下表是霰弹枪清群能力的**下界**。

| 武器 | vs 2 只丧尸 | vs 3 只丧尸 | vs 4 只丧尸 |
|---|---|---|---|
| **自制霰弹枪** | 66.0% | 18.1% | 2.5% |
| 自制猎枪 | 15.0% | 1.0% | 0.0% |
| 手枪 | 12.3% | 0.3% | 0.0% |
| 长剑 | 3.4% | 0.1% | 0.0% |
| 棍棒 | 0.2% | 0.0% | 0.0% |
| 匕首 | 0.9% | 0.0% | 0.0% |

> ⚠ **这些数字是下界，而且低估得厉害**：Arena 把 8 颗全砸在同一只身上——一只无甲丧尸只要 ~16 伤就倒，打进 24 伤是纯浪费（overkill）。霰弹枪真正的清群价值是**一枪扫到多只**，见下表（空间几何量化）。

### 一枪扫到几只（锥形散射的空间几何）

3 只丧尸并排（间距 = 身宽 24px），8 颗弹丸各自在 ±18° 锥内独立采样落点。

| 距离 | 平均命中丧尸数 | 平均命中弹丸数 | ≥2 只被打中的概率 |
|---|---|---|---|
| 20px | 1.00 / 3 | 8.0 / 8 | 0.0 % |
| 30px | 1.00 / 3 | 8.0 / 8 | 0.0 % |
| 45px | 2.02 / 3 | 8.0 / 8 | 77.4 % |
| 60px | 2.62 / 3 | 8.0 / 8 | 97.7 % |

> **散射自己长出了一条距离曲线**（不是设计出来的，是 8 颗独立采样 + 锥角的自然结果）：
> - **贴脸（≤30px）**：锥还没铺开（散布宽 13~19px ＜ 身宽 24px）→ 8 颗**全砸在同一只身上**，24 点伤害一枪撂倒——毁灭性**单体**，不是清群。
> - **中距（45~60px）**：弹丸铺开到能盖住 2~3 只，77~98% 的概率一枪同时咬到两只以上——这才是**清群**档，代价是每只只挨 2~4 颗、且吃距离衰减。
> - **远距（>90px）**：射程外，开不了火。
>
> 玩家因此有一个真实的**站位决策**：想秒掉眼前这只就贴上去，想同时打断三只的逼近就退半步。

## ③ 距离：贴脸毁灭性 vs 拉开只剩零星几颗

弹丸命中数按**几何**估算（引擎不管空间，实弹道在 Godot 层）：±18° 锥在距离 d 处铺开的横向散布宽 ≈ 2·d·tan(18°)，靶宽 = 2×12px（对齐 Actor.Radius）；命中比 = min(1, 靶宽/散布宽)。伤害衰减走 Ballistics.RangedDamageFactor（引擎纯函数）。

| 距离 | 散布宽(px) | 期望命中弹丸数 | 距离衰减系数 | 期望到肉总伤（无甲） |
|---|---|---|---|---|
| 15 | 9.7 | 8.0 / 8 | 100 % | 56.0 |
| 25 | 16.2 | 8.0 / 8 | 92 % | 51.6 |
| 40 | 26.0 | 7.4 / 8 | 76 % | 39.1 |
| 60 | 39.0 | 4.9 / 8 | 53 % | 18.4 |
| 90 | 58.5 | 3.3 / 8 | 20 % | 4.6 |
| 110 | — | **射程外，不可开火** | 0 | 0 |
| 130 | — | **射程外，不可开火** | 0 | 0 |

> 贴脸（≤25px）8 颗全中且满伤；到 90px 只剩 1~2 颗且伤害掉到地板价；>130px 射程外。
