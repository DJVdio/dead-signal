### 一枪扫到几只（锥形散射的空间几何）

3 只丧尸并排（间距 = 身宽 24px），8 颗弹丸各自在 ±18° 锥内独立采样落点。

| 距离 | 平均命中丧尸数 | 平均命中弹丸数 | ≥2 只被打中的概率 |
|---|---|---|---|
| 20px | 1.00 / 3 | 8.0 / 8 | 0.0 % |
| 30px | 1.00 / 3 | 8.0 / 8 | 0.0 % |
| 45px | 2.02 / 3 | 8.0 / 8 | 77.4 % |
| 60px | 2.62 / 3 | 8.0 / 8 | 97.7 % |

> **散射自己长出了一条距离曲线**（不是设计出来的，是 8 颗独立采样 + 锥角的自然结果）：
> - **贴脸（≤30px）**：锥还没铺开（散布宽 13~19px ＜ 身宽 24px）→ 8 颗**全砸在同一只身上**，24 点伤害一枪撂倒——毁灭性**单体**，不是清群。
> - **中距（45~60px）**：弹丸铺开到能盖住 2~3 只，77~98% 的概率一枪同时咬到两只以上——这才是**清群**档，代价是每只只挨 2~4 颗、且吃距离衰减。
> - **远距（>90px）**：射程外，开不了火。
>
> 玩家因此有一个真实的**站位决策**：想秒掉眼前这只就贴上去，想同时打断三只的逼近就退半步。
